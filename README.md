🚀 AI-IDE для работы с DeepSeek
koderDeepS Desktop — это специализированная настольная IDE, которая объединяет встроенный браузер DeepSeek с автоматическим созданием и управлением файлами проекта. Программа автоматически превращает код, сгенерированный DeepSeek, в реальные файлы проекта.

https://screenshot.png

📦 Установка
Клонируйте репозиторий:

bash
git clone https://github.com/yourusername/koderDeepS-Desktop.git
cd koderDeepS-Desktop
Установите зависимости:

bash
pip install PySide6
Запустите программу:

bash
python koderDeepS-Desktop.PY
🎯 Как это работает
Автоматическое создание файлов
DeepSeek генерирует код с указанием имени файла:

text
file: bot.py
print("Hello, World!")
Программа обнаруживает кодовый блок

Автоматически создаётся файл projects/current/bot.py

Пред промпт
Настройте пред промпт через меню Инструменты → Пред промпт

Выберите формат файла кнопкой

При отправке сообщения текст автоматически объединяется:

text
[пред промпт]

[сообщение пользователя]
Сохранение сессии
При первом входе в DeepSeek данные авторизации сохраняются

После перезапуска программы вы остаётесь в аккаунте

📁 Структура проекта
text
koderDeepS-Desktop/
├── koderDeepS-Desktop.PY     # Главный файл программы
├── koderDeepS-Desktop.ico     # Иконка приложения
├── config.json                 # Настройки (пред промпт)
├── projects/                   # Папка с проектами
│   └── current/                # Текущий проект
├── README.md                    # Этот файл
└── requirements.txt             # Зависимости
⚙️ Конфигурация
Настройки сохраняются в config.json:

json
{
  "pre_prompt": "Создай файл Python с именем [имя_файла.py]"
}
